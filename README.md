# 🎓 Udemy Clone

Welcome to my **Udemy Clone** project! This is a responsive frontend website built using **HTML5** and **CSS3**. The project is inspired by the Udemy online learning platform and was created to improve my web development and UI design skills.

## 🚀 Features

* Responsive homepage design
* Modern and clean user interface
* Course cards with images and details
* Navigation bar with search functionality
* Categories section
* Recommended and popular courses
* Footer with useful links
* Built using Flexbox and CSS Grid

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Font Awesome

## 📚 What I Learned

* Creating responsive layouts
* Using Flexbox and CSS Grid effectively
* Structuring webpages with semantic HTML
* Styling modern user interfaces with CSS
* Improving frontend development skills

## 📌 Project Status

This project is completed and was built for learning and practice purposes. It helped me strengthen my HTML and CSS fundamentals by recreating the layout of a popular e-learning platform.
